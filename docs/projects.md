# 项目

## 已完成

### 个人主页

基于 VitePress 搭建的个人主页，适合展示个人介绍、技术笔记和项目入口。

- 技术栈：VitePress、Vite、GitHub Actions、GitHub Pages
- 亮点：响应式首页、自定义主题色、自动部署、按仓库名自动处理 `base`
- 用途：作为个人主页、博客起点或作品集入口

## 下一步准备补充的项目

### 管理后台或中台类项目

适合补充你在权限、表单、图表、复杂交互和组件封装上的经验。

### 数据可视化页面

适合展示你在图表、动效、信息层级和性能优化上的能力。

### 工具型小产品

适合突出你对“从问题出发”的理解，例如 Markdown 工具、表单生成器、文件处理工具等。

## 建议怎么写你的作品集

每个项目建议至少补充这 4 类信息：

1. 项目背景：这个项目解决了什么问题。
2. 你的职责：你负责了哪些模块和关键决策。
3. 技术实现：用了什么方案，为什么这样选。
4. 结果复盘：上线效果、性能指标或业务价值。

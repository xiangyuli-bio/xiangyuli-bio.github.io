# VitePress 主页搭建记录

这篇文章记录了当前个人主页的基础搭建方式，后续你可以在这个结构上继续扩展博客、评论区、访客统计和更多主题样式。

## 当前已经完成的部分

- 使用 `docs` 目录作为站点根目录
- 开启本地搜索
- 配置自定义导航和笔记侧边栏
- 添加 GitHub Pages 自动部署工作流
- 根据仓库名自动处理 `base` 路径，减少部署出错概率

## 本地开发命令

```bash
npm run dev
```

## 构建命令

```bash
npm run build
```

## 发布思路

仓库推送到 `main` 之后，GitHub Actions 会自动执行以下流程：

1. 拉取仓库代码
2. 安装依赖
3. 构建 VitePress 站点
4. 把 `docs/.vitepress/dist` 发布到 GitHub Pages

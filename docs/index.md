## 现在这个主页已经具备什么

- 首页、关于我、项目、笔记、联系页已经就绪。
- `main` 分支推送后会自动部署到 GitHub Pages。
- `base` 路径会根据仓库名自动切换，既支持 `username.github.io`，也支持普通项目仓库。

## 接下来你可以继续补充

- 把首页文案换成你的真实定位和个人介绍。
- 在项目页里加入你自己的截图、链接和成果数据。
- 在 `docs/notes/` 下面持续新增文章，主页就会越来越像一份完整的个人站点。
